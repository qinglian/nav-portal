# 导航门户

一个现代化的导航页面，支持日间/夜间模式切换、编辑模式、本地存储等功能。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 功能

- 🌓 日间/夜间模式切换
- ✏️ 编辑模式 - 添加/编辑/删除网站
- 🔍 搜索功能 (⌘K 快捷键)
- 🖼️ 自动获取网站图标
- 💾 本地数据存储

## 部署

部署到 GitHub Pages：
1. Fork 此仓库
2. 在 Settings → Pages 中选择 Source: GitHub Actions
3. 推送代码即可自动部署

## 技术栈

- React 18
- Vite 5
- Lucide Icons
- CSS Modules
